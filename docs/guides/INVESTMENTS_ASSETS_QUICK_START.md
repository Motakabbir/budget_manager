## ğŸ”¥ Power User Tips

### 1. Multi-Currency Portfolio

- Track investments in their native currency
- USD, EUR, GBP, INR, JPY, AUD, CAD, CHF supported
- Convert to base currency for totals

### 2. Diversification Target

**Recommended**:
- No single type > 30% of portfolio
- Mix high-risk (crypto) with low-risk (bonds)
- Check pie chart regularly

### 3. Update Schedule

**Weekly**:
- Update stock/crypto prices
- Check top/worst performers

**Monthly**:
- Review diversification
- Rebalance if needed
- Update asset values

**Quarterly**:
- Record dividends
- Renew insurance
- Check warranties

### 4. Documentation

- Add notes to each investment/asset
- Why you bought it
- Target sell price
- Important dates
- Tax considerations

### 5. Insurance Optimization

- Set reminders 60 days before expiry
- Compare quotes annually
- Bundle policies for discounts
- Increase coverage for appreciated assets
