# Investments & Assets - Quick Start Guide

**Get started with portfolio tracking in 5 minutes!**

---

## 🚀 Quick Setup

### Step 1: Add Your First Investment (2 minutes)

1. Click **"Investments"** in the sidebar (💰 icon)
2. Click **"Add Investment"** button
3. Fill in the basics:
   - **Type**: Stock (or your investment type)
   - **Name**: "Apple Inc."
   - **Symbol**: AAPL
   - **Quantity**: 10
   - **Purchase Price**: $150
   - **Current Price**: $175
   - **Purchase Date**: Select date
4. Click **"Add Investment"**

✅ **Done!** Your first investment is tracked.

---

### Step 2: Add Your First Asset (2 minutes)

1. Click **"Assets"** in the sidebar (💼 icon)
2. Click **"Add Asset"** button
3. Fill in the basics:
   - **Type**: Vehicle (or your asset type)
   - **Name**: "Tesla Model 3"
   - **Purchase Price**: $45,000
   - **Current Value**: $38,000
   - **Purchase Date**: Select date
4. **For Vehicle** - Add details:
   - Make: Tesla
   - Model: Model 3
   - Year: 2023
5. **Insurance** (optional but recommended):
   - ☑️ Check "Has Insurance"
   - Provider: "State Farm"
   - Policy: "SF-123456"
   - Coverage: $40,000
   - End Date: Select expiry date
6. Click **"Add Asset"**

✅ **Done!** Your first asset is tracked.

---

### Step 3: View Your Analytics (1 minute)

1. Click **"Analytics"** in the sidebar (📊 icon)
2. See your **Net Worth** at the top
3. Explore the **Portfolio Diversification Chart**
4. Check the **ROI Tracker** for performance
5. Try the **Depreciation Calculator**

✅ **Done!** You're analyzing your portfolio.

## 💡 Quick Tips

### Investments

**Update Prices Regularly**:
- Click ⋮ → Edit on investment card
- Update "Current Price" field
- Save to see updated P&L

**Record Dividends**:
- Click ⋮ → Record Dividend
- Enter amount received
- ROI automatically improves

**Track Performance**:
- Green = Profit 📈
- Red = Loss 📉
- Check "Top Performers" for winners

### Assets

**Monitor Insurance**:
- Look for "Needs Attention" section
- Red badges = Expired coverage
- Yellow badges = Expiring soon (< 30 days)

**Track Depreciation**:
- Set depreciation method when adding
- System calculates automatically
- Use calculator on Analytics page

**Update Values**:
- Edit asset to update current value
- Depreciation recalculates
- Keep values accurate for net worth

```
• Other
```

**Quick Add**:
1. Assets page → Add Asset
2. Select type → Fill details → Save
3. Add insurance info (important!)

### Update Investment Price

1. Find investment card
2. Click ⋮ (three dots)
3. Click "Edit"
4. Update "Current Price"
5. Click "Save"

**Result**: P&L updates automatically

### Renew Insurance

1. Check "Needs Attention" section
2. Click on asset with expiring insurance
3. Click ⋮ → "Update Insurance"
4. Enter new end date
5. Save changes

**Result**: Alert disappears

### Record Dividend

1. Receive dividend payment
2. Find investment card
3. Click ⋮ → "Record Dividend"
4. Enter amount
5. Save

**Result**: ROI increases, total dividends updated

### Sell Investment/Asset

1. Find item card
2. Click ⋮ → "Sell"
3. Confirm sale
4. Item marked as sold

**Result**: Historical record preserved

## 🔥 Power User Tips

### 1. Multi-Currency Portfolio

- Track investments in their native currency
- USD, EUR, GBP, INR, JPY, AUD, CAD, CHF supported
- Convert to base currency for totals

### 2. Diversification Target

**Recommended**:
- No single type > 30% of portfolio
- Mix high-risk (crypto) with low-risk (bonds)
- Check pie chart regularly

### 3. Update Schedule

**Weekly**:
- Update stock/crypto prices
- Check top/worst performers

**Monthly**:
- Review diversification
- Rebalance if needed
- Update asset values

**Quarterly**:
- Record dividends
- Renew insurance
- Check warranties

### 4. Documentation

- Add notes to each investment/asset
- Why you bought it
- Target sell price
- Important dates
- Tax considerations

### 5. Insurance Optimization

- Set reminders 60 days before expiry
- Compare quotes annually
- Bundle policies for discounts
- Increase coverage for appreciated assets

## 🆘 Quick Troubleshooting

### "Can't see my investments"
- Check you're logged in
- Refresh page (Ctrl+R)
- Clear browser cache

### "Wrong P&L calculation"
- Update current price
- Check quantity is correct
- Verify purchase price

### "Insurance alert not showing"
- Check end date is entered
- Date must be future (not past)
- Refresh page

### "Form won't submit"
- Check all required fields (red *)
- Verify numbers are positive
- Check date format
